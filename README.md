# AutoInstallAria2
 Auto Install Aria2 To Linux

```
wget "https://raw.githubusercontent.com/Mintimate/AutoInstallAria2/main/AutoInstallAria2ForLinux.sh" && sudo bash AutoInstallAria2ForLinux.sh
```
